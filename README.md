# ⚡ NOISE

Filtra el ruido en salud, nutrición y fitness. Quédate con lo que funciona según la ciencia.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (estilos)
- **Supabase** (auth + base de datos)
- **Stripe** (pagos)
- **Claude API** (IA para análisis)
- **Vercel** (hosting)

## Setup

1. Clona el repo
2. Copia `.env.local` y rellena las variables
3. Instala dependencias:

```bash
npm install
```
